Amazon 大数据服务
=========================

大数据分为以下四个方面，相应的开源工具及AWS服务

##1)聚合器

**开源工具**

+ scribe
+ storm
+ kafka

**AWS**

+ Amazon Kinesis


##2)持续处理

**开源工具**

+ Hadoop 
+ Spark 

**AWS**
+ EMR

##3)数据存储

**开源工具**

+ HDFS
+ HBASE
+ Cascading

**AWS**

+ S3
+ DynamoDB
+ Redshift

##4)分析&报告

**开源工具**

+ Hive
+ Pig
+ Phoenix

**AWS**

+ Redshift (PB级数据)
